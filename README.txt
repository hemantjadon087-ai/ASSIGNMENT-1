C Assignment — Functions and Header Files

This project contains two main objectives:

------------------------------------------------------------
Objective 1 — Function Logic (mylib.h and mylib.c)
------------------------------------------------------------
You will implement the following functions:
- isArmstrong(int num)
- isAdams(int num)
- isPrimePalindrome(int num)

These are declared in mylib.h and defined in mylib.c.  
They are tested using a menu-driven program in main.c.

------------------------------------------------------------
Objective 2 — Array Utilities (arraylib.h and arraylib.c)
------------------------------------------------------------
You will implement common 1D array operations:
- findMaxIndex, findMinIndex, findAverage
- displayArray, reverseArray, sortArray
- linearSearch

These are declared in arraylib.h and defined in arraylib.c.  
They are also tested in main.c.

------------------------------------------------------------
How to Compile and Run
------------------------------------------------------------
Use the following command to compile:
gcc main.c mylib.c arraylib.c -o prog

Then run the program:
./prog

------------------------------------------------------------
Files Included
------------------------------------------------------------
- main.c — Menu-driven program to test all functions
- mylib.h / mylib.c — Functions for Armstrong, Adams, Prime Palindrome
- arraylib.h / arraylib.c — Array helper functions
- README.txt — Instructions for compilation and usage
